# CyberTeck.in - Information Architecture & Content Hierarchy
# Document: 03_INFORMATION_ARCHITECTURE.md

## 1. Content Hierarchy
- **Primary Tier (Awareness & Trust):**
  - Hero with high-fidelity real-time security telemetry components.
  - Social Proof / Trust bars showing leading Indian organizations (SBI, TCS, Tech Mahindra, Airtel, ISRO, NPCI).
  - 10 Security Pillars.
- **Secondary Tier (Value & Validation):**
  - Deep-dive platform pillars: Identity Security, Cloud Security, Threat Intel, AI-SOC.
  - Compliance coverage (DPDP, CERT-In, RBI) mapped to certifications.
- **Tertiary Tier (Conversion & Workspace):**
  - Interactive "Book Assessment" to generate a direct CISO risk score.
  - "CyberTeck OS Dashboard" serving as the live telemetry console.

## 2. Dynamic Data Flows
- User submits Quick Assessment → Triggers score computation on backend → Returns tailored recommendations using Gemini AI API.
- Live Threat Intel stream → Generated dynamically on backend via API simulation → Updates SOC dashboards in real-time.
