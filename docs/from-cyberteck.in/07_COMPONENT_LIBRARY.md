# CyberTeck.in - Component Library Inventory
# Document: 07_COMPONENT_LIBRARY.md

The following enterprise-grade components are built for modularity and accessibility:

1. **Enterprise Header & Mega-Menu (`Navbar`):**
   - Collapsible mobile panel.
   - Expanding product columns detailing our 10 security pillars.
   - Booking/Login indicators.

2. **Legal & Navigation Footer (`Footer`):**
   - Address list, compliance badges.
   - Dynamic navigation grid matching the sitemap.

3. **Posture Risk Gauge (`RiskGauge`):**
   - Animated SVG circle meter charting enterprise-wide posture.
   - Colored states (Red: Critical, Yellow: Moderate, Green: Safe).

4. **Live Feed Panel (`ThreatFeed`):**
   - Real-time scrolling telemetry log displaying simulated events, originating IPs, vectors, and actions.

5. **AI Advisor Console (`AICopilot`):**
   - Scrollable chat panel linked to our Express server proxying Google GenAI models (`gemini-3.5-flash` or `gemini-3.1-pro-preview`).

6. **Compliance Checklist Cards:**
   - Expanding status cards for ISO 27001, DPDP Act, and CERT-In compliance frameworks.
