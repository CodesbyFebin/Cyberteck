# CyberTeck.in - Page Blueprints
# Document: 08_PAGE_BLUEPRINTS.md

## Home Page Layout
1. **Hero Section:** High-impact heading, animated dashboard mock widget (with real-time simulated attacks, compliance dials), search, CTA buttons ("Book a Demo", "Explore Platform").
2. **Trust Bar:** Autoscrolling marquee of Indian enterprise logos.
3. **The Moats:** Grid of 5 structural advantages (AI-First, Compliance-Native, Unified, Trained on Indian Infra, Audit-Ready).
4. **Pillars / Capabilities:** Tabbed panel featuring Identity Security, Cloud Security, Threat Intelligence, and Compliance Automation.
5. **Success Stories:** Dynamic slider showing SBI, TCS, and Mahindra success cases.

## Dashboard Layout
1. **Header Workspace:** Dynamic greeting, current risk dial, active alert counts.
2. **Left Panel:** Metric grids, live event timelines, interactive checklists.
3. **Right Panel:** AI Copilot chat interface connected to Gemini, ready to answer questions.

## Book Assessment Layout
1. **Form Step Flow:** Multi-step question path collecting company size, compliance needs, active audits.
2. **Assessment Logic:** Computes dynamic scoring immediately on submission and uses Gemini to analyze results and output a PDF-style report.
